zope.viewlet Package Readme
===========================

Overview
--------

Viewlets provide a generic framework for building pluggable user interfaces.

Changes
-------

See CHANGES.txt.

Installation
------------

See INSTALL.txt.


Developer Resources
-------------------

- Subversion browser:

  http://svn.zope.org/zope.viewlet/

- Read-only Subversion checkout:

  $ svn co svn://svn.zope.org/repos/main/zope.viewlet/trunk

- Writable Subversion checkout:

  $ svn co svn+ssh://userid@svn.zope.org/repos/main/zope.viewlet/trunk

- Note that the 'src/zope/viewlet package is acutally a 'svn:externals' link
  to the corresponding package in the Zope3 trunk (or to a specific tag, for
  released versions of the package).
